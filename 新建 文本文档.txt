接口名称：评论/回复（后端已完成）
接口路径：/api/comment
请求方式：POST
参数：content,userId,productId,rootCommentId
接口说明：用户的评论功能
参数
后端需要接收的信息:
{
    "content": //评论内容
    "userId": //所属用户id
    "productId": //所属商品id
    "rootCommentId": //父评论id（为空说明该评论没有父评论）
    "toCommentId"://回复的评论id（如果为空说明该评论没有回复其他评论）
    
}
然后后端将对应的评论添加到数据库中


前端接收的消息：(前端只需要知道后端是否添加数据成功即可，失败的话则评论发送失败)
{
  "code": 200,
  "message": "成功",
  "data":{
     "productId"：...
     "content": //评论内容
    "rootCommentId": //父评论id（为空说明该评论没有父评论）
    "toCommentId"://回复的评论id（如果为空说明该评论没有回复其他评论）
    "sendTime"：...//发送时间
  }
}
接口名称：获取商品所有信息，包括评论(后端已完成)
接口路径：/api/commentarea
请求方式：GET
请求参数：productId
接口说明：后端向前端返回productId对应的商品的所有信息,包括其对应的评论。
{
     "code":200,
     "message":"获取成功",
     "data":
     {
         userId:...,
         productName:...,
         price:...,
         producer:...,
         salesVolumn:...,
         productImg:...,
         surplus:...,//surplus为剩余量
         productComment：
         [
         {
            productCommentId:...,
            content:...,
            sendTime:...,
            userId:...,//发布该条评论的人的id
            commentLikeCount:...,
         }
         ]         
     }
}
接口名称：获取该评论的所有子评论
接口路径：/api/comment/childcomment
请求方式：GET
请求参数：product_comment_id
接口说明：后端向前端返回product_comment_id对应的所有子评论,即所有root_comment_id为该product_comment_id的评论
{
     "code":200,
     "message":"成功返回子评论"，
     "data":
     {
           productCommentId:...,//该子评论的id
           toCommentId：...该评论回复的评论的评论id
           content:...,
           sendTime:...,
           userId:...,//该子评论的发出者
           commentLikeCount:...//点赞数
           
     }
}
